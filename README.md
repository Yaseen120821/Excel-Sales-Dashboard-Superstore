# Excel-Sales-Dashboard-Superstore

This project is an interactive Business Intelligence Dashboard created in Microsoft Excel, built using the Superstore Sales dataset.
The goal of this project is to transform raw data into meaningful business insights using Pivot Tables, KPI metrics, What-If Analysis, and Scenario Manager.

🚀 Project Outcomes

🔹 Designed a fully automated Sales Performance Dashboard
🔹 Enabled interactive filtering using Slicers and Timeline
🔹 Built KPI cards to monitor key business metrics
🔹 Created Pivot Tables & Pivot Charts to visualize sales trends
🔹 Performed What-If Analysis & Scenario Manager to forecast business outcomes
🔹 Added Insights & Recommendations to support strategic decision-making

📌 Dashboard Features

Feature	Description
KPI Metrics	Total Sales • Profit • Customers • Orders
Slicers	Category • Segment • Region • Ship Mode
Timeline Filter	Order Date (Month & Year)
Visuals	Column/Bar Charts • Line Chart • Pie Chart
What-If Analysis	Goal Seek — target sales forecasting
Scenario Manager	Best Case (+10% Sales) & Worst Case (-10% Sales)
Automation	Refresh Dashboard Button
